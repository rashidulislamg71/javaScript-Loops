### For Loop
Syntax:
for (initialization; condition; increment / decrement) {
    // code block to be executed repeatedly
}

for (let i = 1; i <= 5; i++) {
  console.log(i);
}
এখানে কী হচ্ছে?
let i = 1 → শুরু ১ থেকে
i <= 5 → ৫ পর্যন্ত চলবে
i++ → প্রতি বার ১ করে বাড়বে

# While Loop
While Loop এর সাথে একটি শর্ত যুক্ত থাকে। যতক্ষণ পর্যন্ত শর্ত টি সত্যি (true) থাকবে, ততক্ষণ লুপ চলতেই থাকবে।

সতর্কতা: শর্ত যদি কখনও মিথ্যা না হয়, তবে লুপটি থামবে না (Infinite loop), যা ব্রাউজার হ্যাং করে দিতে পারে।
Syntex:   
             while (condition) {
   	        // যে কোড বারবার চলবে
 }
	For Example:
		let i = 1;
while (i <= 5) {
   		      console.log(i);
   		       i++;
}

Syntax এর ৩টা অংশ
Initialization → let i = 1
Condition → i <= 5
Update → i++

নোটঃ 
When to use while loop? 
-কতবার চলবে আগে জানা না থাকলে। 
-iteration সংখ্যা fixed না হলে.

do...while loop 
do...while loop এমন একটা loop
 যেটা condition false হলেও কমপক্ষে ১ বার চলবেই
সহজ কথা
আগে কাজ → পরে condition check
Syntex:
do {
    // এই কোড আগে চলবে
} while (condition);
 কাজ করার ধাপ
 আগে do ব্লকের কোড চলে
 তারপর condition check হয়
condition true হলে আবার loop
false হলে stop
For Example: 
let i = 10;
do {
    console.log(i);
} while (i < 5);

নোটঃ 
When to use do..while loop? 
যখন condition check করার আগে একবার লুপ চালানোর প্রয়োজন।













Break & Continue:

জাভাস্ক্রিপ্ট লুপে break এবং continue হলো ট্রাফিক সিগন্যালের মতো। এরা লুপের স্বাভাবিক গতিপথ পরিবর্তন করে দেয়। চলুন খুব সহজে এদের কাজ বুঝে নিই।

Break (থেমে যাওয়া)
break মানে হলো "থামা!"। যখনই লুপের ভেতর break কার্যকর হয়, লুপটি সাথে সাথে বন্ধ হয়ে যায় এবং লুপের বাইরে থাকা পরবর্তী কোড চলতে শুরু করে।
বাস্তব উদাহরণ: ধরুন, আপনে ১০ তলা একটি বিল্ডিংয়ের লিফটে আছেন। আপনার টার্গেট ৫ তলা। ৫ তলায় পৌঁছালে লিফট ব্রেক করবে(লুপ ব্রেক করবে); আপনে লিফট থেকে নেমে যাবেন (loop execute লুপ থেকেবেড়িয়ে যাবে) , বাকি তলাগুলোতে লিফট গেলেও তুমি আর তাতে নেই।
Example Code:
for (let floor = 1; floor <= 10; floor++) {
    if (floor === 5) {
        console.log("টার্গেট ফ্লোর ৫ এসে গেছে! নামছি...");
        break; // লুপটি এখানেই শেষ
    }
    console.log("লিফট এখন ফ্লোর: " + floor);
}
// আউটপুট: ১, ২, ৩, ৪ পর্যন্ত প্রিন্ট করে ৫ এ এসে থেমে যাবে।

Continue (একটি স্টেপ বাদ দিয়ে পরেরটিতে যাওয়া)
continue মানে হলো "এটা বাদ দাও, পরেরটা দেখো!"। এটি লুপকে পুরোপুরি বন্ধ করে না, বরং বর্তমানের কাজটুকু বাদ দিয়ে লুপের পরবর্তী ধাপে (Next iteration) চলে যায়।
বাস্তব উদাহরণ: আপনে বাজারে গেছো ৫টি জিনিস কিনতে। লিস্টের ৩ নম্বর জিনিসটি দোকানে নেই। তখন ধরুন, আপনে ওটা বাদ দিয়ে ৪ নম্বর জিনিসটি খুঁজতে শুরু করলে। ৩ নম্বর আইটেমটি তুমি skip করলে। 
Example Code:
for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        console.log("৩ নম্বর আইটেম নেই, স্কিপ করছি!");
        continue; // ৩ প্রিন্ট না করে সরাসরি ৪ নম্বরে চলে যাবে
    }
    console.log("আইটেম চেক করছি: " + i);
}
// আউটপুট: ১, ২, (৩ নেই), ৪, ৫


for, while, break এবং continue ব্যবহার করে Problem Solving:
১. জোড় সংখ্যার মেলা (The Even Number Quest)
টার্গেট: ১ থেকে ২০ পর্যন্ত সব সংখ্যা প্রিন্ট করো। শর্ত: শুধুমাত্র জোড় (Even) সংখ্যাগুলো প্রিন্ট হবে। বেজোড় সংখ্যা আসলে continue ব্যবহার করে সেটাকে স্কিপ করতে হবে।

২. চোর ধরা (Catch the Thief)
টার্গেট: একটি অ্যারে আছে: ["Ali", "Rohim", "Chor", "Karim", "Jobbar"] শর্ত: লুপ চালিয়ে নামগুলো চেক করো। যখনই "Chor" পাওয়া যাবে, তখন break দিয়ে লুপ বন্ধ করে দাও এবং কনসোলে দেখাও "চোর ধরা পড়েছে!"। চোরের পরের নামগুলো আর চেক করার দরকার নেই।
৩. বিজোড় সংখ্যার যোগফল (Sum of Odds)
টার্গেট: ১ থেকে ৫০ পর্যন্ত সংখ্যাগুলোর মধ্যে যেগুলা বিজোড়, সেগুলার যোগফল বের করো। শর্ত: for লুপ ব্যবহার করবে। যদি সংখ্যাটি জোড় হয়, তবে continue ব্যবহার করে তা এড়িয়ে যাবে। শেষে মোট যোগফল প্রিন্ট করবে।
৪. শপিং ডিসকাউন্ট (Discount Checker)
টার্গেট: তোমার কাছে কিছু পণ্যের দামের অ্যারে আছে: [100, 250, 500, 1200, 750, 3000, 900] শর্ত:
লুপ চালিয়ে দামগুলো চেক করো।
যদি কোনো পণ্যের দাম ১০০০ টাকার বেশি হয়, তবে সেটি কেনা সম্ভব না (স্কিপ করো)।
যদি মোট খরচ ২০০০ টাকা ছাড়িয়ে যায়, তবে কেনাকাটা বন্ধ করো (break) এবং মেসেজ দাও "বাজেট শেষ!"।
৫. আনলাকি নাম্বার (The Unlucky 13)
টার্গেট: ১ থেকে ২০ পর্যন্ত সংখ্যা প্রিন্ট করো। শর্ত: কিন্তু যখনই ১৩ আসবে, সেটি প্রিন্ট না করে একটি মেসেজ দিবে "১৩ একটি আনলাকি নাম্বার, স্কিপ করছি!" এবং লুপ চালিয়ে যাবে।
৬. পাসওয়ার্ড ট্রায়াল (Password Limit - Advanced)
টার্গেট: while লুপ ব্যবহার করে একটি সিস্টেম বানাও। শর্ত:

৫ বার ভুল করলে "Account Blocked" মেসেজ দাও।
For…..in & for…..of Loop:
জাভাস্ক্রিপ্টে for...in এবং for...of লুপ দুটি দেখতে প্রায় একরকম হলেও এদের কাজ সম্পূর্ণ আলাদা। সহজে বোঝার উপায় হলো: in কাজ করে ইনডেক্স বা প্রপার্টি নিয়ে, আর of কাজ করে সরাসরি ভ্যালু বা উপাদান নিয়ে।

for...in লুপ (Object)
এটি মূলত কোনো অবজেক্টের Key বা Property বের করার জন্য ব্যবহার করা হয়। এটি অ্যারের ক্ষেত্রেও কাজ করে, তবে তখন এটি উপাদানের বদলে ইনডেক্স (Index) নাম্বার দেয়।
const user = {
    name: "Sakib",
    age: 25,
    city: "Dhaka"
};
for (let key in user) {
    console.log(key + " : " + user[key]);
}
আউটপুট: 
// name : Sakib
// age : 25
// city : Dhaka

মনে রাখবে: for...in লুপ অবজেক্টের প্রপার্টির নাম (key) গুলো প্রদান করে।
for...of লুপ (Array)
এটি সরাসরি কোনো লিস্ট বা অ্যারের ভেতরকার উপাদান (Value) গুলো নিয়ে কাজ করে। এটি অবজেক্টের ওপর সরাসরি কাজ করতে পারে না। কারণ অবজেক্ট ইটারেবল নয়।
const colors = ["Red", "Green", "Blue"];

for (let color of colors) {
    console.log(color);
}

 আউটপুট:
// Red

প্রবলেম: আপনার কাছে ১০ জন স্টুডেন্টের মার্কস আছে। for...of ব্যবহার করে তাদের গড় (Average) বের করো। যদি কোনো স্টুডেন্ট ১০০-তে ১০০ পায়, তবে তাকে "Special Mention" দিয়ে লুপটি কন্টিনিউ করো।



উপসংহার (Conclusion)
পরিশেষে বলা যায়, প্রোগ্রামিংয়ের জগতে Loop হলো আপনার এমন এক বিশ্বস্ত সহকারী, যে ক্লান্ত না হয়ে আপনার দেওয়া হাজার হাজার কাজ মুহূর্তেই করে দিতে পারে। আপনি যখন লুপের ব্যবহার রপ্ত করবেন, তখন বড় বড় গাণিতিক সমস্যা সমাধান করা বা ডেটা প্রসেসিং করা আপনার জন্য একদম সহজ হয়ে যাবে।
মনে রাখবেন, লুপ শেখার আসল মন্ত্র হলো প্র্যাকটিস। ওপরের প্রবলেমগুলো নিজে সমাধান করার চেষ্টা করুন, কোড রান করুন এবং ভুল হলে বারবার চেষ্টা করুন। ভুল থেকেই আপনি শিখবেন কখন for ব্যবহার করতে হয় আর কখন while লুপের সাহায্য নিতে হয়।
লুপ আয়ত্ত করতে পারলে আপনি জাভাস্ক্রিপ্টে আরও জটিল লজিক তৈরিতে আত্মবিশ্বাসী হয়ে উঠবেন। তাই আজই আপনার কোড এডিটর খুলুন আর লুপের ম্যাজিক শুরু করুন!
